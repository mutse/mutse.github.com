编写spec文件

切换至SPEC目录下，并编写hello.spec文件，内容如下：

Summary: Hello World rpm package
Name: hello
Version: 1.0
Release: 1
License: GPL
Packager: mutse
Group: Application
URL: http://mutse.tk

BuildRoot: %{_builddir}/%{name}
AutoReqProv: no

%description
This is a software for rpm package.

%prep
%build
%install

%pre
%post
chmod 755 /usr/local/bin/hello

%files
/usr/local/bin/hello
